What I'm focused on

My recent work spans two main areas: improving how AI coding agents operate safely and efficiently, and building complete web applications end to end.

On the tooling side, I build the infrastructure that lets coding agents work autonomously in isolated, secure environments, including sandboxed containers, scoped authentication, and reusable skill definitions that carry over between projects and agents. On the product side, I design and maintain full-stack applications, handling everything from user interfaces to backend services, data, and deployment.

  • Agent infrastructureSecure, sandboxed environments and scoped credentials so agents can work independently and safely.

  • Full-stack productsComplete applications shipped to real users, with automated testing and deployment pipelines.
  • Portable AI skillsReusable skill and workflow definitions that work across different AI coding assistants.
